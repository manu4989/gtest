Printing Graduation Certificates

Over the years it has become a somewhat error prone process when it is time to print the certificates given to the students as they graduate. As such, this guide is intended to help streamline that process as much as possible.

Open the database manager application and select the students receiving certificates.

Choose the Print icon from the database manager application.

Properly load the certificate paper in the printer.(This is often where most of the issues arise)
