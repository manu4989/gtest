The Output Presets

When you look at the Outptut Presets you will find several.of them bearing descriptive names that indicate their intended use.

Responsive HTML5 - Coffee This is the default or primary layout. All information is included and it uses the Indigo Coffee layout. Upon generating and opening, it may prove a bit confusing without knowing the scope and intent of this project.
Responsive Intranet Site Intended to create an output suitable for uploading to a company intranet where only information intended for employees and students will be seen. This output uses Dynamic Content Filters to allow controlling the Table of Contents and Index so that only topics applicable to Students or Administrators is shown. The output uses the Intranet_Site_TOC as the Table of Contents.
Responsive Intranet Site Alternate This is the same information as the one above, but using an alternate version of the skin.
Responsive Web Site Intended to create an output suitable for uploading to a web server and presenting content as any other customer facing web site does. This output uses Dynamic Content Filters to allow controlling the drink menu between Hot and Cold drinks. The output uses the Web_Site_TOC as the Table of Contents.
Responsive Web Site This is the same information as the one above, but using an alternate version of the skin.
WebHelp Intranet Site This is a WebHelp output that would be suitable for uploading to a company intranet where only information intended for employees and students will be seen.
WebHelp Web Site This is a WebHelp output that would be suitable for uploading to a web server and presenting content as any other customer facing web site does.
Microsoft HTML Help This is an output that will produce a Microsoft HTML Help (CHM) file. The content has been limited to Student content.
Printed Documentation This output should create a PDF that would be used for printing the content of the Student guide.
