The Parking Topic

Click here to view the live topic described by this topic.

One of the new features of RoboHelp 2019 is the ability to work with images in the SVG format. The Parking topic is used to illustrate this aspect. You will see a diagram of the parking lot that has been created using the SVG format.

Look at the images section in the RoboHelp Reimagined project to learn more about working with SVG.
