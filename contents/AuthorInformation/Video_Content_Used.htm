Video Content Used

In preparation of this sample project, some video content was created using the barista school theme. The smallest video file is included with the project and the others have been uploaded to different video sharing sites. (YouTube, Vimeo, and ScreenCast.com).

The topic named Tuesday (Module 4) demonstrates using a local video that is part of the file set of this sample.

The topic named Thursday (Module 1) demonstrates presenting two videos. One is a video that was uploaded to the Vimeo.com video sharing site and a second video uploaded to the ScreenCast.com video sharing site.

The topic named Monday (Module 4) demonstrates presenting a video that was uploaded to the YouTube video sharing site.
