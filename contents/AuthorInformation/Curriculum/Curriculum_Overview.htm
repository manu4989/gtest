The Curriculum Overview Topic

Click here to view the live topic described by this topic.

As the project is about a barista school, it seems only natural that a curriculum should be presented. This topic outlines the six week process.

Each bullet point is also a Dynamic HTML (DHTML) drop-down that may be clicked to reveal hidden information. Within each of the hidden areas is a hyperlink pointing to the About topic that outlines the agenda for each of the weeks.

Because the same overview information is actually used in two different topics, the information has been inserted into a Snippet and the Snippet added to each of the two topics.

The topic also uses the links feature and the bullets feature of RoboHelp.

Open  the RoboHelp Reimagined project for more information on creating links, applying bullets, using snippets and Dynamic HTML Drop-Downs.
