The Captivate Folder

This folder exists over in the Main Project folders and stores all the Captivate HTML 5 output files. There are two different Captivate projects used in this sample. One is for the Splash Banner as seen on many web sites where five images are presented in a slide show fashion. The second is a quiz to evaluate whether someone has the traits that would make for a good barista. Each folder contains a collection of files and more folders that are related to the Captivate HTML 5 output.

Note that the opening page presented when you generate some of the outputs loads up the Splash page. And that page has a META redirect in the HTML code that points to the Captivate created banner.)

The second Captivate project containing the quiz appears inside an Inline Frane. The Inline Frame is configured to point to the Index.html of the Captivate output files.

Check out the Inline Frame and Captivate information in the RoboHelp Reimagined project.
