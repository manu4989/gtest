The Splash Topic

Click here to view the live topic described by this topic.

When a RoboHelp output is created, there is one topic that is nominated to be the "Default Topic". This is the first topic presented to the viewer when the system loads up.

In this project the topic is named "Splash".

Essentially this topic contains a META redirect code in the HTML that points to the start page for one of the Captivate outputs. In order to see the redirect, look at the HTML code of the Splash.htm topic. This also serves to demonstrate one possibility of what can be created using Adobe Captivate and combined with Adobe RoboHelp.
